# PortEdu - Portfólio Educacional

O **PortEdu** é um modelo de Portfólio Educacional digital, desenvolvido em HTML e CSS puro, projetado para ajudar educadores e estudantes a organizar, documentar e refletir sobre seu desenvolvimento profissional e acadêmico.

## Estrutura do Projeto

O projeto é composto por cinco arquivos HTML principais e um arquivo CSS centralizado para o estilo:

| Arquivo | Descrição |
| :--- | :--- |
| `index.html` | Página inicial e visão geral do portfólio. |
| `artefatos.html` | Seção dedicada à listagem e visualização de projetos, trabalhos e documentos. |
| `competencias.html` | Seção para acompanhar o desenvolvimento de competências-chave. |
| `rubricas.html` | Seção que detalha os critérios de avaliação e rubricas. |
| `metacpd.html` | Seção para definir e monitorar Metas de Desenvolvimento Profissional Contínuo (CPD). |
| `style.css` | Folha de estilos centralizada para manter a consistência visual em todas as páginas. |

## Melhorias Implementadas

Com o objetivo de tornar o projeto **100% funcional e pronto para uso** como um modelo de portfólio estático, as seguintes melhorias e correções foram implementadas:

1.  **Separação de Estilos (Refatoração):**
    *   O CSS embutido em cada arquivo HTML foi extraído e consolidado em um único arquivo, `style.css`.
    *   Isso melhora a **manutenibilidade**, **consistência visual** e **desempenho** do projeto.
2.  **Consistência de Navegação (Correção de Bug):**
    *   O menu de navegação (`<nav>`) foi padronizado em todas as páginas para usar a mesma estrutura de cabeçalho (`<header>`) e estilos do `style.css`.
    *   Os links de navegação foram corrigidos para apontar para os arquivos HTML corretos em todas as páginas.
3.  **Estrutura da Página Inicial (`index.html`):**
    *   O conteúdo da página inicial foi refatorado para servir como uma **página de boas-vindas** e **direcionamento**, com links claros para as seções principais.
4.  **Aprimoramento de Conteúdo:**
    *   O conteúdo das páginas secundárias foi expandido e estruturado em tabelas e cards para melhor visualização e organização das informações, tornando o modelo mais completo.

## Como Usar

Para visualizar o portfólio, basta abrir o arquivo `index.html` em qualquer navegador web.

O projeto agora está **100% funcional** como um modelo de portfólio estático, com uma estrutura de código limpa e consistente.
